rst_autogenerate¶

sphinx_docs.rst_autogenerate.generate_rst_for_specified_module(file_location, module_name, rst_output_location)¶

Generates an .rst file for a specified module, using the given module name and output location.

Parameters:

  • file_location – The file path of the Python module. This is used to ensure the file exists.

  • module_name – The importable name of the module, as it should be referenced in the documentation.

  • rst_output_location – The directory path where the .rst file should be created.
